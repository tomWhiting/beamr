# B-029: Utility stubs — logger, unicode, sys, gleam_stdlib identity

> **Cluster:** beamr
> **Depends on:** B-026
> **Checklist:**
> - C150 — BIF registry maps (Module, Function, Arity) to a Rust function pointer
> **Stories:**

## Purpose

gleam_otp actor and supervisor import utility functions from OTP modules. Implement as native Rust stubs.

## Task

Implement logger:warning/2, unicode:characters_to_list/1, unicode:characters_to_binary/1, sys:debug_options/1, gleam_stdlib:identity/1 in stdlib_stubs.rs.

## Requirements

### R1: logger:warning/2 and gleam_stdlib:identity/1

logger:warning/2 writes warning to stderr. gleam_stdlib:identity/1 returns its argument unchanged.

**Acceptance:**
- logger:warning(Format, Args) writes to stderr
- logger:warning with bad format is no-op
- gleam_stdlib:identity(X) returns X

**Files:**

### R2: unicode conversions

unicode:characters_to_list/1 and characters_to_binary/1 handle UTF-8 conversion. Binary passthrough for already-UTF-8 content.

**Acceptance:**
- characters_to_binary(<<hello>>) returns <<hello>>
- characters_to_list(<<abc>>) returns [97,98,99]
- characters_to_binary([104,105]) returns <<hi>>
- Invalid input returns error tuple
