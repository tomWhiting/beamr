# B-040: Gleam stdlib wrappers — second wave and sample workflow e2e

> **Cluster:** beamr
> **Depends on:** B-037, B-038, B-039
> **Checklist:**
> - C150 — BIF registry maps (Module, Function, Arity) to a Rust function pointer

## Purpose

The gleam_stdlib module contains 29 Erlang FFI wrapper functions that Gleam's standard library delegates to. With B-037 through B-039 dependencies in place, implementing these wrappers unblocks the sample_workflow:run/1 end-to-end test — the milestone that proves beamr can execute a real Gleam workflow.

## Task

Implement 29 gleam_stdlib BIFs, 2 uri_string stubs, and un-ignore the sample_workflow_e2e test to verify end-to-end execution.
