# B-082: Group leader read/write BIFs

> **Cluster:** beamr
> **Depends on:** B-016 (process spawning)
> **Checklist:**
> - C150 — BIF registry maps (Module, Function, Arity) to a Rust function pointer

## Purpose

Every BEAM process has a group leader — a PID that receives I/O output. `io:format` sends messages to the group leader rather than writing to stdout directly. OTP's `init` and `application` modules read and set group leaders during startup. Without these BIFs, OTP code that manages I/O routing fails.

## Task

Add a group_leader field to Process and implement the `erlang:group_leader/0` and `erlang:group_leader/2` BIFs.

## Boundaries

- SHALL NOT implement full I/O protocol message routing — group_leader is stored but not used for I/O dispatch (beamr uses IoSink from B-039)
- SHALL NOT implement group leader inheritance across spawn — spawned processes get the parent's group leader (simple copy)
- SHALL return the process's own PID as default group leader if none is set

## Requirements

### R1: Group leader field on Process

Add a `group_leader: Term` field to Process, defaulting to the process's own PID.

**Acceptance:**
- Process gains `group_leader: Term` field
- Default group_leader is the process's own PID (set at spawn)
- Spawned processes inherit the parent's group_leader
- Test: spawn process, verify group_leader is parent's PID

**Files:**
- modify: crates/beamr/src/process/mod.rs

**Checklist:** C150

### R2: group_leader/0 and group_leader/2 BIFs

Register `erlang:group_leader/0` (returns current group leader) and `erlang:group_leader/2` (sets group leader for a process).

**Acceptance:**
- `group_leader()` returns the calling process's group_leader PID
- `group_leader(NewLeader, Pid)` sets the group leader for Pid, returns true
- Setting group leader for a non-existent PID returns badarg
- Test: get default group leader, set a new one, verify the change

**Files:**
- create: crates/beamr/src/native/process_info_bifs.rs (add to existing if B-080 creates it, otherwise create)
- modify: crates/beamr/src/native/mod.rs

**Checklist:** C150
